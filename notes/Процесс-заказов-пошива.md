# Процесс заказов пошива — регламент

> Дата: 15.07.2026 | Утверждён: Нурлан

## Правило
Каждый заказ пошива проходит согласование через **Навигатора**.

## Этапы
1. 🔵 Поступление заказа → **Продажник** уведомляет Навигатора
2. 🟡 Расчёт себестоимости → **Производство + Финдир** передают данные Навигатору
3. 🟢 Одобрение → **Навигатор** проверяет и утверждает
4. ⚪ Внесение в таблицы → ТОЛЬКО после одобрения Навигатора
5. 🟣 Запуск в работу → **Производство** подтверждает старт
6. 🔴 Готово → уведомление Навигатору

## Ключевые моменты
- Никаких прямых правок в таблицах без одобрения
- Все цифры (себестоимость, маржа, сроки) согласовываются
- Каждый шаг фиксируется в этом Obsidian vault
