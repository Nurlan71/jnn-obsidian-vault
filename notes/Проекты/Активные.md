# Активные проекты

> Текущие проекты JNN — статус, сроки, ответственные

## Как вести список

Формат записи:
```
### [Название проекта]
- **Статус:** 🟢 В работе / 🟡 Пауза / 🔴 Проблема / ✅ Завершён
- **Ответственный:** [[Имя]]
- **Срок:** ДД.ММ.ГГГГ
- **Описание:** краткое описание
- **Ссылка:** [Google Drive](url) / [Sheets](url)
```
