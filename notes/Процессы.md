# Процессы JNN

## Продажи
1. Лид → Продажник
2. Переговоры → Закрытие сделки
3. Заказ → Производство
4. Контроль → РОП

## Производство
1. Получение заказа → Производство
2. Швейный цех / Печатный цех
3. Контроль качества
4. Отгрузка

## Маркетинг
1. Стратегия → Маркетолог
2. Контент → Тимур + Айгерим
3. Реклама → Аналитика
4. Лиды → Продажник

## Финансы
1. Учёт → Финансист + Алина
2. Бюджеты → Контроль
3. Зарплаты → Начисление
4. Налоги → Отчётность
