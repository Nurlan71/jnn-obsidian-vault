# Настройка Google Cloud для JNN агентов

## Шаг 1: Создай проект в Google Cloud

1. Открой https://console.cloud.google.com
2. Вверху нажми на выпадающий список проектов → **Новый проект**
3. Название: `jnn-agents`
4. Нажми **Создать**

![Создание проекта](step1.png)

---

## Шаг 2: Включи нужные API

1. В левом меню: **API и сервисы** → **Библиотека**
2. Найди и включи:
   - **Google Drive API**
   - **Google Sheets API**
   - **Gmail API** (опционально)

![Включение API](step2.png)

---

## Шаг 3: Создай сервисный аккаунт

1. В левом меню: **IAM и админ** → **Сервисные аккаунты**
2. Нажми **Создать сервисный аккаунт**
3. Название: `jnn-agents`
4. Описание: `AI агенты JNN для работы с Google Drive и Sheets`
5. Нажми **Создать**
6. Пропусти шаг с правами (или дай "Редактор")
7. Нажми **Готово**

![Сервисный аккаунт](step3.png)

---

## Шаг 4: Создай JSON-ключ

1. Кликни на созданный сервисный аккаунт `jnn-agents`
2. Перейди на вкладку **Ключи**
3. Нажми **Добавить ключ** → **Создать новый ключ**
4. Выбери формат **JSON**
5. Нажми **Создать**
6. Файл скачается автоматически

![Создание ключа](step4.png)

---

## Шаг 5: Передай ключ

**Вариант A (рекомендуется):**
Положи скачанный файл на сервер:
```
/opt/data/credentials/google-service-account.json
```

**Вариант B:**
Пришли мне файл в Telegram.

---

## Шаг 6: Дай доступ сервисному аккаунту

1. Скопируй email сервисного аккаунта (выглядит как `jnn-agents@jnn-agents.iam.gserviceaccount.com`)
2. Открой Google Drive
3. Для каждой папки/таблицы, к которой нужен доступ:
   - ПКМ → **Поделиться**
   - Добавь этот email как **Редактора**

![Доступ](step6.png)
