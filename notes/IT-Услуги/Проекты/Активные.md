## Чеклист проекта

Для каждого нового проекта:
- [ ] УТП согласовано с клиентом
- [ ] Дизайн-макет утверждён
- [ ] Контент получен (тексты, фото, контакты)
- [ ] Техническая часть готова
- [ ] Тест на телефоне и компьютере
- [ ] Ссылка передана клиенту
- [ ] Инструкция передана
- [ ] Оплата получена
- [ ] Отзыв получен
