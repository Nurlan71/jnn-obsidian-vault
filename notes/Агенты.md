# AI-Агенты JNN

## Структура

| Агент | Роль | Получатели в Telegram |
|-------|------|----------------------|
| **Navigator (OWL)** | COO / Оркестратор | Нурлан |
| **Habit Control** | Продуктивность | Нурлан |
| **Маркетолог** | CMO | Нурлан + Айгерим1 |
| **РОп** | Sales Director | Нурлан + Айгерим2 |
| **Продажник** | Менеджер продаж | Нурлан + Руслан |
| **Финансист** | CFO | Нурлан + Алина |

| **HR-директор** | HR | Нурлан |

## Принципы
- Dashboard — главный интерфейс
- Telegram — только уведомления
- Все решения через Нурлана
- Ежедневные отчёты в 20:00
- Координация через Navigator

## Технологии
- OpenRouter / owl-alpha (default)
- NVIDIA Nemotron (vision)
- TTS: edge-tts
- Хранилище: сервер + Google Drive + Obsidian vault
