# JNN Obsidian Vault — Назначение

## Зачем нужен репозиторий

jnn-obsidian-vault — это **центральное хранилище знаний** JNN Factory. Служит для:

1. **Документации** — все регламенты, инструкции, процессы компании
2. **Заметок агентов** — результаты работы, отчёты, решения
3. **Синхронизации** — доступ с любого устройства через Obsidian + GitHub
4. **Бэкапа** — история изменений через git
5. **Совместной работы** — агенты и сотрудники могут редактировать документы

## Структура

- `01 Проекты/` — активные проекты
- `02 Архитектура/` — техническая архитектура
- `02 Задачи/` — текущие задачи
- `03 Отчёты/` — отчёты агентов
- `03 Производство/` — документы производства
- `04 Команда/` — информация о команде
- `Habit/` — трекер привычек
- `landings/` — лендинги
- `notes/` — заметки по отделам (Финансы, HR, Маркетинг, Проекты, Агенты, Команда, Инфраструктура)
