# Obsidian Vault — JNN

База знаний швейного производства JNN.

## Структура

- `Главная.md` — точка входа
- `notes/` — все заметки
- `attachments/` — вложения (изображения, файлы)
- `templates/` — шаблоны заметок

## Синхронизация

Vault синхронизируется через Git. Каждое изменение автоматически коммитится.

## Как использовать

1. Открой папку `/opt/data/obsidian-vault` в Obsidian
2. Редактируй заметки — изменения автоматически попадут в Git
3. Для совместной работы — push/pull репозитория

## Связи между заметками

Используй синтаксис `[[Имя заметки]]` для создания ссылок между заметками.
