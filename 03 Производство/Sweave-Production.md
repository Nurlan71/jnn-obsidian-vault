# Швейное производство JNN — Учёт выработки

## Структура оплаты швей

- **Ежемесячная оплата** — фиксированный оклад
- **Еженедельная оплата** — сдельная часть за выработку
- **Ежедневный учёт** — сколько каждая швея отшила

## Что нужно отслеживать

1. **Швея** — ФИО, должность, дата найма
2. **Продукция** — тип изделия, расценка за единицу
3. **Выработка** — количество отшитых изделий за день/неделю
4. **Оплата** — оклад + сдельная часть
5. **Статус** — активна / уволена / отпуск

## Связь с другими агентами

- **Производство** → **Финдир**: данные о выработке → Финдир начисляет зарплату
- **HR** → **Производство**: приём/увольнение швей → Производство обновляет списки
- **Производство** → **Продажник**: готовность заказов → Продажник отгружает
